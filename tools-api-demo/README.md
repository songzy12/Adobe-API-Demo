# Tools API

## Node.js

Install [Node.js 10.13.0](https://nodejs.org/en/download/) or higher.

```bash
nvm ls

nvm uninstall 8
nvm install 12
nvm use 12
nvm alias default node

nvm current
```

```bash
npm init
npm install --save @adobe/documentservices-pdftools-node-sdk
```
